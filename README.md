## Run Locally

Clone the project

```bash
  git clone https://github.com/GymSquad/NYCULib-Web-Archive.git
```

Go to the project directory

```bash
  cd NYCULib-Web-Archive
```

Install dependencies

```bash
  npm ci
```

Start the server

```bash
  npm run dev
```
